## Backstory
Back in 2018, I saw a video about Windows remote shutdown, which peaked my interest and led me to learn CMD commands and about Windows in general. After some time, I got bored of CMD and the Batch scripting language and started to learn and use Python instead. The next evolutionary phase was discovering that a WiFi network could be cracked or DOSed with a few lines of commands, which once again, got my interest. When I was researching how to do the WIFI attacks I saw on Youtube, it got apparent that every single tutorial needed a Linux-based system to work. That made me learn how to use Linux and after learning to use it on a basic level, I finally managed to hack my own WIFI network. The feeling of successfully exploiting the WiFi weakness was amazing, and it gave me motivation to study ways different devices could be exploited. The rest is history...
